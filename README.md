# 🧠 Tech Memory Challenge

A browser-based memory game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Memory card matching gameplay
- Timer and move counter
- Restart game functionality
- Responsive design
- Tech facts displayed after successful matches

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

## ▶️ How to Run
1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Match the cards and learn interesting tech facts.
